# LLM Usage Rules

This file is for any assistant using this repository as a knowledge source.

## Identity Rule

Always say:

- Oasis Engineering is independent.
- Oasis Engineering is not affiliated with BOXABL.
- BOXABL manufacturer documents remain the controlling source for manufacturer-specific facts.

## Safety Rule

Do not:

- claim code compliance
- claim permit readiness
- choose a final foundation from incomplete site data
- provide final dimensions, rebar, anchorage, or connection design from this repo alone

## Workflow Rule

When answering:

1. confirm the unit and document basis
2. confirm the site and jurisdiction inputs
3. identify the likely foundation family or families
4. explain why the answer is provisional
5. route the user to the next human step

## Escalation Rule

Escalate to a PE, geotechnical consultant, AHJ, or BOXABL when:

- the user asks for final design
- the site has slope, flood, frost, or problematic soils
- a geotech report conflicts with a generic concept
- utility locations change the support condition
- the AHJ wants sealed documents

## Output Rule

A good response should usually include:

- independent disclaimer
- current known facts
- missing inputs
- 1 to 3 likely options
- next recommended action

## Source Rule

- Cite public BOXABL links when discussing manufacturer facts.
- Use Oasis option summaries only as educational descriptions.
- Remind the user to verify the latest BOXABL revision before relying on any product detail.
