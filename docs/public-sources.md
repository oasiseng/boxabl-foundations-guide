# Public Sources

The canonical source registry — including every URL and its last verification date — is machine-readable at [`data/SOURCES.yaml`](../data/SOURCES.yaml). A scheduled CI job ([check-sources](../.github/workflows/check-sources.yml)) probes every URL weekly and fails loudly when one dies.

This page is the human-readable summary.

## BOXABL

- [BOXABL Home](https://www.boxabl.com) — high-level current public lineup snapshot.
- [BOXABL How It Works](https://www.boxabl.com/how-it-works) — public process framing around feasibility, site prep, delivery, and setup.
- [BOXABL Technical Documentation](https://www.boxabl.com/technical-docs) — public gateway for official technical references.
- [BXB-000009 Installation Manual](https://gcdn.boxabl.com/documents/technical/plan-sets/BXB-000009%20Install%20Manual.pdf) — primary source for Studio Casita dimensions, shipping size, weight, R-values, and foundation-scope language. Note: the manual states the approved drawings control over the manual in any discrepancy.

- [Baby Box 120 Spec Sheet (BXB-000029)](https://gcdn.boxabl.com/documents/technical/plan-sets/Baby%20Box%20120%20Spec%20Sheet%20(BXB-000029).pdf) — primary source for Baby Box RV facts.

URL housekeeping note: links to BOXABL PDFs are kept clean (no analytics query strings) and standardized on the `gcdn.boxabl.com` host, which was verified reachable. If BOXABL migrates hosts, update `data/SOURCES.yaml` first.
