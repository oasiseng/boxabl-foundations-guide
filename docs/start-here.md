## Core Principle

The safe public answer is usually:

- explain the issue
- identify the missing project inputs
- name the likely foundation paths
- state what must be verified by BOXABL, the AHJ, the geotechnical report, and the PE of record

The unsafe answer is:

- choosing a final foundation from incomplete data
- claiming code compliance
- giving permit-ready dimensions, reinforcing, anchorage, or connection design from public summaries
